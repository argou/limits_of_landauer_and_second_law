# limits_of_landauer_and_second_law
Limits of Landauer Principle and the Second Law of Thermodynamics. Contains Python scripts and documentation

📌 Overview
This repository explores the fundamental limits of the Landauer principle and of the second law of thermodynamics through numerical simulations.
It includes Python scripts that model different one-bit memory systems and analyze their thermodynamic behavior.
📂 Contents

memory_type1.py – Simulation of the first memory model.
memory_type2.py – Simulation of the second memory model.
memory_type3.py – Simulation of the third memory model.
docs/explanatory_document.md – Detailed explanation of the models and theoretical background.

🧠 Purpose
These simulations aim to illustrate:

The energy cost of information erasure (Landauer principle).
The difference between statistical entropy and information entropy.
Examples of violation of the second principle at nanoscale
